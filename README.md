## 🌟 Featured Project: Lery AI

> **A hybrid IoT + AI ecosystem that teaches English without the anxiety of being judged.**

Most language apps fail for the same reason: they run on the device people are most distracted by (the phone), and they correct users in real time, triggering what researchers call the *Affective Filter* — the psychological wall that blocks fluency.

**Lery AI** flips the model. A screen-less smart speaker becomes the conversation partner. The phone becomes the dashboard you check *after* the session. Mistakes are dignified, not interrupted.

🧠 **Architecture highlights:**
- **Hardware:** Raspberry Pi running a Finite State Machine (IDLE → LISTENING → THINKING → SPEAKING → ERROR), with local wake-word detection via OpenWakeWord (ONNX, runs fully offline for privacy and zero-latency activation).
- **AI Pipeline:** Whisper for STT → Gemini 1.5/3.1 Flash as the conversational orchestrator with strict system prompts → ElevenLabs for natural TTS. Audio is processed in volatile memory and deleted immediately — LGPD-compliant by design.
- **Backend:** Fastify 5 + TypeScript + Prisma + PostgreSQL 15, structured around `Language → Level → Module → Lesson` with dynamic per-lesson prompts.
- **Pedagogy as code:** CEFR-aligned scoring (A1–C2) across four pillars — Task Achievement, Grammatical Accuracy, Vocabulary, Fluency. A 70% threshold gates progression; below 50% locks the lesson; between 51–69% triggers a review module.
- **GradeHITL framework:** Students can dispute scores → a secondary LLM Reflector agent re-evaluates and recalibrates the primary evaluation prompt, mitigating algorithmic bias around accents and intelligibility.
- **Scaffolding & Code-Switching:** When the learner freezes, Lery temporarily switches to Portuguese to explain a structure, hands them the English equivalent, and gently guides them back ("Let's try again in English!").

A project where **product, hardware, AI pipelines, and pedagogy converge** — the kind of system I love to build.

## 💭 To whoever is reading this

If you're a developer scrolling through profiles wondering whether you'll ever *get there* — I want you to know something:

Three years ago I was where you are. No portfolio, no certainty, just curiosity and the stubborn belief that good systems can make people's lives better. Every line of code on this profile started as a problem I didn't know how to solve. Every AI pipeline I now design started with one prompt that didn't work.

**The gap between where you are and where you want to be isn't talent. It's reps.**
